# Game

- Start with a tile in the center of the screen. Tile activates when you click on it
	- Tiles have a cooldown before being able to be clicked again
	- Clicking gives you some RESOURCE
- When you have ~20 RESOURCE, you can buy an "auto-clicker". You can place it freely, but you should place it next to the RESOURCE tile to trigger it automatically

- Cool animation and sound effect for each trigger
- "tubes" between connected tiles
- Drag and drop

## Type of game

- Incremental automation game
- Kind of like factorio but without the player character
- Kind of like "The Farmer was Replaced", but less programmy and more block based
- Kind of like "scritchy scratchy" but less "manual labor" and more thoughtful placement of blocks
- Kind of like Stacklands, but without cards, and instead a meta progression system/skill tree
- More than one currency type (wood, stone, gold, iron, coal, etc)
- Kind of like "keep on mining", but less "magical" in the sense that lightning comes down from above, and more placement of blocks/chains
- Less finding resources on a map and more building production chains etc., like growing flowers and then harvesting them. Maybe fruits instead? Kind of like the gnome hole fruit game, but isometric

- Meta progression
